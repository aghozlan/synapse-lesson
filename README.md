## 🚀 Overview
In response to the educational challenges in Lebanon (March 2026), this application was developed to provide students with a high-fidelity, **asynchronous learning experience**. It transforms the complex biological process of the synapse into a cinematic, 3D interactive lesson.

## 🛠 Features
* **Cinematic 3D Engine:** Built with `Three.js` for stylized neon visuals that make molecular biology "crystal clear."
* **Scroll-Linked Animation:** Students control the timeline of the nerve impulse through intuitive scrolling.
* **AI Rule-Based Tutor:** An integrated tutor that "locks" the lesson until the student demonstrates understanding of key concepts (e.g., Calcium ion triggers).
* **Mobile & Desktop Friendly:** Lightweight code-generated geometry ensures performance on various devices.

## 📖 The 5E Strategy Applied
1.  **Engage:** A glowing axon terminal invites the student to start the impulse.
2.  **Explore:** Students navigate the 3D space to find the "Calcium Gates."
3.  **Explain:** The AI Tutor provides immediate feedback on neurotransmitter release (Exocytosis).
4.  **Elaborate:** Visual transitions show the chemical-to-electrical signal conversion.
5.  **Evaluate:** (Upcoming) Final assessment module to track student progress.
